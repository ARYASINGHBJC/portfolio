Game Development

Multiplayer Ping-Pong

Ping pong built using React, Node.js, Express.js, and Socket.io is a web-based multiplayer game of ping pong. React is a JavaScript library used for building user interfaces, Node.js is a JavaScript runtime environment, Express.js is a web application framework, and Socket.io is a library that enables real-time, bidirectional communication between web clients and servers. In this game, React is used to create the user interface and handle user interactions, Node.js and Express.js are used to run the server and handle requests and responses, and Socket.io is used to enable real-time communication between the game clients and the server.

Technologies used include:

  • React
  • Node.js
  • Express.js
  • Socket.io
  • MongoDB
